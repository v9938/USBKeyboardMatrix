# MSX KEYBOARD MATRIX サブ基板  対応機種

*の機種は、各種資料からの推測になり、実機での動作・取り付け未確認の機種になります。
## ■Panasonic
### JIGU_FS-A1.zip         
備考：Resetキー機能には、非対応  

    対象機種：[MSX2] FS-A1  

### JIGU_FS-A1mk2.zip  
備考：Resetキー機能には、非対応  

    対象機種：[MSX2] FS-A1mk2  

### JIGU_FS-A1F.zip  
備考：PAUSEキーが現状上手く動作しません。（原因調査中）

    対象機種：[MSX2] FS-A1F  
    (FS-A1FMは、配置は同じだがコネクタピッチが異なるため対応不可)

    対象機種：[MSX2+]  FS-A1FX*/FS-A1WX/FS-A1WSX  
             [MSX TR] FS-A1ST/FS-A1GT
